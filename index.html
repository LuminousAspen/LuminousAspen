about us

MayhemCraft is a community-driven Minecraft server dedicated to providing players with a fun and exciting gaming experience. Our server offers a semi-vanilla Minecraft experience for all interested. Thanks to the 'quality of life' plugins you'll be able to set warps/homes, /tpa your friends, make teams and more. It's a brand new edition with a spawn in a charming cherry blossom forest. Come build this world together!





rules

1. Follow Vortex of Mayhem's Discord rules.
2. Don't abuse glitches, do not hack or use macros.
3. Don't grief, don't steal, don't build innapropriate builds.
4. Don't harass other players, leave them alone if asked to.
5. Don't PVP without the other player's consent.
6. Don't purposefully cause server lag.
7. Do not find loopholes in the rules to exploit them, use common sense.
8. If you find a glitch, report it to the staff on discord.
9. And most importantly, have fun!





server

This server is comptaible with both Java (cracked) and Bedrock.
